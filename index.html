Introduction to Concurrent Programming with Stackless Python

This is a work-in-progress draft of a tutorial I'm writing on Stackless Python. It is available in html and pdf formats. Code is included in the text, and in a standalone gzipped archive. Most examples should run as-is on a working install of Stackless Python 2.4.3, except for the samples in the chapter on actors, which require that pyGame be installed for the visualization engine.
